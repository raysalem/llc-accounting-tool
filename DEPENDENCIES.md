# Connectivity & Dependencies

This project relies on the following environment and libraries:

## Software Requirements
- **Node.js**: Version 14.x or higher is recommended.
- **Microsoft Excel**: Required to view and manually edit the accounting data.

## Node.js Dependencies
These are automatically installed when you run `npm install`:

1.  **exceljs** (`^4.4.0`): The core library used for reading, writing, and manipulating Excel (`.xlsx`) files.
2.  **fs** (Built-in): Used for file system operations like checking if files exist.
3.  **path** (Built-in): Used for managing file paths across different operating systems.

## Installation
To install the necessary dependencies, run the following command in the project root:

```bash
npm install
```
